Blorbioteca is a free and open source character and world repository.
What's a character / world repository?

A character / world repository lets artists organize details and art for their original characters and settings. Typically this includes basic details like name and species, more detailed information like backstory, and artwork.

What does free and open source mean?

Free software isn't about price, it's about your ability to use, modify, and share software. While non-free software attempts to restrict modifications and how it's used, free software encourages the user to use, alter, and share the software. Open source means that the source code for this software is freely available.

This means if you want to create your own version of Blorbioteca or change parts of the code to suit your needs, you can!

Blorbioteca's source code is hosted on GitHub and licensed under version 3 of the GNU General Public License. This license allows for modification and distribution of software, with the following stipulations:

  • All shared versions of the software, modified or not, must be licensed under the GPLv3
  • Modified versions should be clearly indicated as such
  • All shared versions of the software must make the source code available to the user.
How can I contribute to Blorbioteca?

If you want to help contribute to the website's code base, check out our GitHub. Bugs and errors should also be opened as an issue on the GitHub. If you have feedback or suggesstions, use the contact page.

Terms and Conditions

Blorbioteca is intended for human made original characters and settings. As such, the following is not allowed on the site:

  • Art and text produced with generative AI
  • Character sheets of real people
  • Art and content that glorifies any form of bigotry, including but not limited to racism (including nazism), sexism, homophobia, transphobia, religious discrimination, and violence against animals and children

To keep the site safe and enjoyable for all, we require users to properly tag artwork and character / world sheets. Generally, more detailed tagging is better, but only a few categories are required to be tagged. For artwork, any visual presence of the following requires tagging. For text based content, detailed description of any of the following requires tagging. Repeated failure to properly tag the following may result in your account being frozen or terminated:

  • Light nudity - artwork where nipples are visible
  • Full nudity - artwork where genitalia is visible
  • Sexual content - depictions or descriptions of sexual scenes
  • Body horror - unnatural distortions to the body
  • Gore - visible guts, gratuitous amounts of blood, nail / eye / teeth trauma, cannibalism, and pregnancy complications
  • Self harm - violence against oneself
  • Child harm - violence against children
  • Animal harm - violence against animals
  • Drug use - visible insobriety, pills, syringes, etc.
  • Eye strain - images containing very saturated / bright / neon colors, strobing / flashing lights, or quick glitching animations
  • Trypophobia - multiple holes close together
  • Arachnophobia / entomophobia - detailed / realistic drawings of bugs, swarms of bugs, unnaturally large bugs, etc.
